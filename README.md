# Website

Recueil de notes concernant l'architecture Event-Driven et les microservices.

Ce site est construit avec [Docusaurus](https://docusaurus.io/).

## Déploiement

Le site est automatiquement déployé sur Azure Static Web Apps via GitHub Actions lors des push sur la branche `master`.

### Problèmes de déploiement

Si vous rencontrez des erreurs avec le workflow GitHub Actions, consultez le [Guide de dépannage](TROUBLESHOOTING.md) pour des explications détaillées et des solutions.
